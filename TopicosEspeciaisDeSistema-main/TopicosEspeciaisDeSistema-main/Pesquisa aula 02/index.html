Porque é importante usar gerência de configuração?

A gerência de configuração de software é de extrema importancia para um grupo de desenvolvimento para que o trabalho não seja perdido durante o desenvolvimento,

O que é integração contínua em um projto de software?

A integração continua é quando o desenvolvedor atualiza o código alterado no projeto principal com amesma frequencia que as funcionalidades são criadas, deixando sempre o código atualizado para o resto da equipe

Quais ferramentas podem ser usadas para verificar débito técnico?

Ferramentas que buscam a organização do projeto como Kanban,

Quais ferramentas podem ser usadas para realizar integração contínua?

Existem algumas ferramentas que podem ser usadas para realizar essa integração, sem que as atualizações sejam perdidas e trabalho seja disperdiçados, são as principais Jenkins e Sonar

O que é DevOps

DevOps é a união de pessoas, processos e tecnologias para dar suporte para e equipe de desenvolvimento. Permitindo que funções anteriormente isoladas, como operações de TI, engenharia de qualidade, segurança, etc, trabalhem de forma cordenada para no final gerar um produto de melhor qualidade

Qual o relacionamento da integração contínua com o conceito de DevOps?

DevOps permite que todas as partes de um time trabalhem como uma coisa só, enquanto as ferramentas da integração continua organiza o ambiente de trabalho
